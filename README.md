### 프로젝트 1. 뮤직 스토어
![](musicstore.png)

### 설명
애플뮤직 스토어에서 음악 리스트를 검색하고 이를 갖고옵니다.<br />
갖고온 음악 리스트 중 하고자 하는 목록을 클릭하면 자동으로 내 음악 리스트에 추가됩니다.<br />
한 번 추가한 항목은 그 다음 실행에서도 확인할 수 있습니다.<br />
<br />
<br />

### 프로젝트 2. 트렌드 코리아
![](trendkorea.png)

### 설명
구글 트렌드 RSS의 데이터를 활용해 현재 한국의 트렌드를 한 눈에 볼 수 있습니다.<br />
버튼을 클릭시 해당 뉴스로 페이지를 이동합니다.<br />

# 실행방법
![](select.png)
